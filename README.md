# EduTech Brief KR (에듀테크 브리프 KR)

국내 교육 및 AI 산업 관련 언론 보도를 수집하여 제공하는 뉴스 브리핑 서비스입니다.

## 1. 프로젝트 실행 (로컬 환경)

브라우저에서 `index.html` 파일을 열면 바로 실행됩니다.
(탐색기에서 더블클릭 또는 VS Code Live Server 활용)

## 2. 매일 뉴스 업데이트 방법

### 2-1. 수동 업데이트 (지금 바로 갱신하고 싶을 때)
프로젝트 폴더 내에 있는 **`update.bat`** 파일을 더블 클릭하세요.
구글 뉴스(RSS)에서 최신 데이터를 가져와 `data.js`를 갱신합니다.

### 2-2. 자동 업데이트 (GitHub 연동 시)
이 프로젝트를 GitHub에 Push하면, 내장된 **GitHub Actions** 스크립트가 작동합니다.
- **실행 시간**: 매일 한국 시간 **오전 7시** (UTC 22:00)
- **작동 방식**: 클라우드 서버가 자동으로 뉴스를 갱신하고 Repository에 Commit합니다.
- **확인 방법**: GitHub Pages로 배포된 웹사이트에서 아침마다 확인 가능합니다.

## 3. 데이터 구조 편집
뉴스의 내용은 `data.js` 파일에 저장되어 있습니다.
- `category`: 기사 카테고리 (policy, local, edutech, trend)
- `importance` / `insight`: 실무자용 요약 및 시사점

> **참고**: 자동 업데이트(`update_news.js`)는 무료 RSS 피드를 사용하므로, 샘플 데이터처럼 정교한 요약/시사점을 제공하지 못할 수 있습니다. (추후 OpenAI API 연동 권장)
